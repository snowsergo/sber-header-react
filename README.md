## Установка проекта

Скопируйте проект на компьютер:
```
git clone https://github.com/snowsergo/sber-header-react.git
```
Установите зависимости проекта:
```
yarn install
```
Запуск в dev режиме:
```
yarn start
```
Финальная сборка:
```
yarn build
```
